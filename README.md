# Projet_S8

## Installation

1.  **Cloner le dépôt :**
    ```bash
    git clone https://github.com/AxeRaoRt/Projet_S8.git
    cd Projet_S8 
    ```

2.  **Installer les dépendances :**
    Il est recommandé d'utiliser un environnement virtuel (comme conda ou venv).
    ```bash
    pip install -r requirements.txt
    ```
